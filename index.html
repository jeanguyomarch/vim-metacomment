vim-metacomment

Installation

  • With Vundle
    • Add Bundle 'jeanguyomarch/vim-metacomment' in your $HOME/.vimrc
    • Re-run vim and type :BundleInstall
  • As a standalone plugin
    • Put doc/vim-metacomment.txt in $HOME/.vim/doc
    • Put plugin/vim-metacomment.vim in $HOME/.vim/plugin
    • Run vim and type :helptags ~/.vim/doc

Usage

  • For a detailed usage type :help vim-metacomment in vim
